💬 Dialogues
Scene 1 — Asking someone's name
A
请问，你叫什么名字？
Qǐngwèn, nǐ jiào shénme míngzi?
Excuse me, what's your name?
B
我叫李文。你呢？
Wǒ jiào Lǐ Wén. Nǐ ne?
My name is Li Wen. What about you?
A
我叫白家月。
Wǒ jiào Bái Jiāyuè.
My name is Bai Jiayue.

📝 Grammar Points

1. Basic Sentence Order: Subject + Predicate + Object

Chinese follows the same basic order as English: Subject → Verb → Object.

Subject + 叫 + Name
我叫李文。
Wǒ jiào Lǐ Wén.
My name is Li Wen.
你叫什么名字？
Nǐ jiào shénme míngzi?
What is your name?

2. Negation with 不 (bù)

Place 不 before a verb to make it negative.

我 + 不 + 是 + ...
我不是李文。
Wǒ bú shì Lǐ Wén.
I am not Li Wen.

3. Apology Pattern

对不起 → 没关系 / 没事
A: 对不起！ B: 没关系。
A: Duìbuqǐ! B: Méi guānxi.
A: Sorry! B: It's okay.

4. Chinese Name Structure: Surname + Given Name

李文 → 李 + 文
Lǐ Wén → Lǐ (surname) + Wén (given name)
Li = family name, Wen = first name
